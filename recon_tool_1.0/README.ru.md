# Recon Tool – Пассивный сбор данных

Автоматизированный инструмент для пассивного сбора информации о веб-приложениях и доменах. Находит поддомены, проверяет живые хосты, собирает исторические URL и формирует чистый отчёт в форматах HTML и JSON.

**Важно:** Используйте только на доменах, на которые у вас есть явное письменное разрешение владельца. Неправомерное использование – ваша ответственность.

---

## Возможности

- **Пассивный сбор поддоменов** через публичные источники (DNS, сертификаты, архивы).
- **Проверка живых хостов** с определением статуса HTTP/HTTPS, технологий и заголовков.
- **Сбор исторических URL** из архивов Wayback Machine и Common Crawl.
- **Генерация отчёта** в двух форматах:
  - `report_<домен>.json` – машиночитаемые данные,
  - `report_<домен>.html` – визуальный отчёт с таблицами и статистикой.
- **Асинхронное выполнение** – параллельные запросы экономят время.
- **Автоматическая установка зависимостей** – необходимые инструменты загружаются при первом запуске.

## Установка

### Системные требования
- Linux (рекомендуется Ubuntu/Debian, работает на любом дистрибутиве)
- Необходимые пакеты: `git`, `curl`, `wget`, `python3`, `python3-pip`, `python3-venv`, `golang-go`

### Установка Python-зависимостей в виртуальном окружении
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

(Опционально) Если вы предпочитаете установить Go-утилиты вручную:

    subfinder

    assetfinder

    amass

    httpx

    waymore (устанавливается через pip install waymore)

Все бинарники должны находиться в $PATH или в папке tools/.
Запуск скрипта с доменом
bash

python main.py example.com

Или без аргументов – вас попросят ввести домен в интерактивном режиме.
Структура проекта
text

Настройка параметров

Все настройки сосредоточены в core/runner.py. Вы можете изменять:

    Количество потоков – в аргументах httpx (-threads).

    Задержку между запросами – в httpx через -delay.

    Таймауты – в вызове run_command() для каждой утилиты.

    Источники для waymore – можно добавить дополнительные провайдеры через флаги.

Пример: чтобы увеличить скорость, поставьте -threads 20 и уберите -delay (но убедитесь, что вы соблюдаете правила вашей программы).
Пример отчёта

После выполнения скрипта вы получите:

    report_example.com.json – все собранные данные в структурированном JSON.

    report_example.com.html – красивый HTML-отчёт со сводкой, таблицами поддоменов и живых хостов.

Откройте HTML в браузере – он содержит всё необходимое для ручного анализа.
